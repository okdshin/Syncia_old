Syncia
================
Syncia is P2p servant application using Neuria.  

Required
----------------
* cchace  
* boost (version >= 1.50)  
* gcc (version >= 4.7.1)  
* jsoncpp  

Running Sample
----------------
You can compile and execute Syncia with below line  
make -f Servant.mk && ./Servant.out 54321
